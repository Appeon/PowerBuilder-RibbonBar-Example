# Testing with Web API Tester

If you created a Web API project in SnapDevelop, there is a built-in Web API tester for testing Web APIs locally to check requests and responses during development.

Let's see how to use the Web API tester to send an HTTP request to a Web API and then check the response.

1. Open the controller file that contains the Web API to be tested. 

2. Right mouse click in the code block of the Web API method, and select Run Test(s). 

   ![](images\webapi\webapi_button.bmp)

3. Wait a short while. The Web API Tester will be launched.  

   ![](images\webapi\webapi_tester.bmp)

4. If required, enter parameter values for the Web API, and then click the Send button to send an HTTP request to the provided URL. The response will be displayed below with the data and response status.

   ![](images\webapi\webapi_tester.bmp)

5. You can select a different Web API from the left panel and execute HTTP request to the Web API.
